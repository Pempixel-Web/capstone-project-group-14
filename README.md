# Capstone Project Group 14

## Quick Summary

This project is a React-based web application built for a capstone group assignment. It features a responsive UI with reusable components for a hero section, navbar, planet cards, planet table, video section, contact form, and footer.

Key features:
- Responsive page layout using React components
- Planet data presentation with cards and table view
- Embedded video section and contact form
- Clean design with modular CSS for each component

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open the app in the browser at `http://localhost:3000`

## Project Structure

- `src/` — application source code
- `src/components/` — reusable UI components
- `public/` — static public assets

## Notes

This project was created with Create React App and uses standard React scripts for development, testing, and build.
